# Toko Madura Management System (TMMS) 🏪

Sistem Manajemen Toko dan Kasir Modern yang dirancang khusus untuk efisiensi operasional toko retail, warung, atau UMKM. Aplikasi ini fokus pada kemudahan penggunaan dengan tampilan yang bersih, modern (Bento Grid), dan performa yang cepat.

## 🚀 Fitur Utama

- **Dashboard Performa**: Ringkasan penjualan, total piutang pelanggan, dan peringatan stok menipis dalam satu layar.
- **Sistem Kasir (POS)**: Antarmuka kasir yang intuitif, mendukung pencarian barang cepat, manajemen keranjang, dan pilihan metode pembayaran (Tunai/Hutang).
- **Manajemen Stok Produk**: Kontrol penuh atas inventaris barang, kategori, harga modal vs jual, serta sistem otomatis deteksi stok kritis.
- **Sistem Piutang (Bon)**: Pencatatan hutang pelanggan yang terintegrasi dengan data pelanggan dan riwayat transaksi.
- **Laporan & Analisis**: Visualisasi data penjualan (Omzet) dalam berbagai periode (Hari, Minggu, Bulan, Tahun, atau Custom Tanggal).
- **Analisis Laba Rugi**: Perhitungan otomatis Laba Kotor (Gross Profit) dan Laba Bersih (Net Profit) setelah dikurangi biaya operasional.
- **Manajemen Biaya**: Fitur pencatatan pengeluaran operasional toko (Listrik, Gaji, Transport, dll).
- **Ekspor Data**: Kemudahan untuk mengunduh laporan penjualan dalam format CSV untuk pembukuan eksternal.

## 🛠️ Spesifikasi Teknis

- **Framework**: React 18 + Vite
- **Styling**: Tailwind CSS (Utility-first CSS)
- **Visualisasi Data**: Recharts (Chart & Graphics)
- **Animasi**: Framer Motion / Motion (Transition & Interaction)
- **Ikon**: Lucide React
- **Persistence**: LocalStorage API (Data tersimpan di browser secara lokal)
- **UI Design**: Modern Responsive Design, Bento-grid layout, Mobile-friendly.

## 📦 Startup & Development

1. Install dependensi:
   ```bash
   npm install
   ```
2. Jalankan server pengembangan:
   ```bash
   npm run dev
   ```
3. Build untuk produksi:
   ```bash
   npm run build
   ```
